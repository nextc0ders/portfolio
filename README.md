# Portfolio (адаптировано)

Проект основан на репозитории https://github.com/CommunityPro/portfolio-html и адаптирован под личные учебные и практические нужды.

Я использовал исходный шаблон как отправную точку и внёс следующие изменения:
- Добавлены новые страницы и компоненты.
- Улучшена структура CSS и утилитарные классы.
- Адаптировано под локальную разработку и учебные эксперименты.

Структура проекта (основное):
- `index.html` — главная страница
- `assets/` — статические ресурсы (иконки, манифест и т.д.)
- `components/` — переиспользуемые шаблоны (`header`, `footer`)
- `css/` — стили проекта
- `js/` — скрипты
- `pages/` — дополнительные страницы (блог, контакты, проекты)

Как запустить
- Открыть `index.html` в браузере (проект — статический сайт).
- Для локальной разработки можно использовать любой простой HTTP-сервер, например:

```bash
# Python 3
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

Авторство и цель

- Оригинальная база: CommunityPro/portfolio-html (ссылка выше).
